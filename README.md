# Semaine-01-BAGOGA-Abdoul-kader-ASSOGBA-Rodrigue-S.O.S-Signal-Distress

## Equipe- BAGOGA-Abdoul-kader- ASSOGBA-Rodrigue

## Description

The first objective of the project is to translate the universal distress signal SOS into sound and light signals using Morse code. The second objective is to translate any word into Morse code, and then into sound and light signals.

## Materials Used
- Arduino Uno
- 1 LED rouge
- 1 resistance 220 Ohm
- Breadboar
- jumper
